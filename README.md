This project is a **comprehensive RESTful API for an E-commerce system**, built with **Java Spring Boot**.  
It includes **most of the core services of real-world e-commerce platforms**, such as:

- User management  
- Product catalog and categories  
- Cart and order management  
- Online payment via **VNPAY**

The API is designed to be **secure, scalable, and easy to integrate**.
